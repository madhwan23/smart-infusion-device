
# Smart Infusion Device

This project automates IV bottle replacement using:
- Liquid level sensor
- Servo motor
- Arduino

## Features
- Detects empty bottle
- Automatically rotates next bottle
- Reduces manual workload

## How to Run
1. Upload code to Arduino
2. Connect sensor and servo
3. System runs automatically
## Hardware Code
The Arduino code (arduino_code.ino) controls:
- Servo motor for bottle switching
- Sensor input for liquid level detection
## How to Run (Simulation)
1. Open in GitHub Codespaces or Replit
2. Run:
   python main.py

## Hardware Implementation
- Upload arduino_code.ino using Arduino IDE
- Connect:
  - Sensor → Pin 2
  - Servo → Pin 9

## Project Type
- Simulation + Hardware Integrated System
